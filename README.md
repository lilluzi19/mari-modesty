<h2>Mari Modesty</h2>

<p>A clean and responsive website for an abaya clothing brand. It showcases products, brand story and customer-facing content with a polished luxury feel.</p>

<h2>Tech Stack</h2>

<ul>
  <li>HTML.</li>
  <li>CSS.</li>
  <li>JavaScript.</li>
  <li>PHP.</li>
  <li>MySQL.</li>
  <li>PHPMyAdmin.</li>
  <li>Bootstrap.</li>
  <li>Figma.</li>
  <li>Visual Studio Code.</li>
</ul>

<h2>Features</h2>

<ul>
  <li>Fully responsive e-commerce storefront with desktop, tablet, and mobile layouts.</li>
  <li>Interactive shopping basket with quantity controls, live totals, and persistent cart data.</li>
  <li>Wishlist functionality for saving favourite products across browsing sessions.</li>
  <li>Multi-step checkout flow with custom form validation and order summary.</li>
  <li>Product search, filtering, and category-based browsing for faster product discovery.</li>
  <li>Light/Dark theme toggle with saved user preference and system theme detection.</li>
  <li>Smooth scrolling, animated UI interactions, and a responsive navigation experience.</li>
  <li>Interactive product galleries, sliders, and promotional carousels.</li>
  <li>Mobile-first navigation with touch-optimised menus and gestures.</li>
  <li>Optimised loading experience with custom animations, preloader, and responsive performance enhancements.</li>
  <li>Real-time user feedback through custom notifications, validation messages, and interactive UI states.</li>
  <li>Client-side storage using Local Storage and Session Storage to preserve user preferences and shopping data.</li>
</ul>
